Image Preview

See, what you search

Image Preview renders a screenshot for every searchresult, so you can see the site

Security:

Because the previews are images, no data is loaded upon searching

Because you know, what waits for you, you can decide if it is the right result

Use Image Preview

  1. Search something
  2. Go into the url bar
  3. Change https://opensearch-mu.vercel.app/search?q=... to https://opensearch-mu.vercel.app/imagesearch?q=...
  4. Loading might take some time because the API is a bit slow
